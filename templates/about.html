How to use the tool?


If you have the Uniprot Accession Code

Simply click on the "Uniprot Accession Code" tool link in the navbar and enter your Uniprot Accession Code into the text field and click submit. The page will then display the results of the matched motifs, the positions and the amount of matches.


If you have the FASTA format code

Simply click on the "FASTA Format" tool link in the navbar and enter your Fasta Format Code into the text field and click submit. The page will then display the results of the matched motifs, the positions and the amount of matches.

How it works?


Once the user has inputed their submission (e.g. Uniprot Accession Code or the FASTA format code, then the software programmed with Python will run a regular expression on this. If the submission is via a Uniprot code then the Uniprot server will be contacted and the relevant FASTA code for the Uniprot code will be retrieved. If an error occurs due to contacting the server then the page is refreshed for the user to reattempt data entry.

The regular expression is as follows...
[^P][^PKRHW][VLSWFNQ][ILTYWFN][FIY][^PKRH]
Key: ^ - Means to not include the following character, [] - Means any of the included characters within the brackets

The resulting matches are then ran through a for loop to discover their positions and add the motif and positions to a dictionary to later be output via a table.
